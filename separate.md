#graph-theory 

For a subset X of [[vertex|vertices]] and [[edge]]s of G and two vertex sets A,B in G we say that X separates A and B if each A-B-[[path]] contains an element of X.

Note that if X separates A and B, then necessarily $A∩B ⊆X$.
