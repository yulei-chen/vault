#theorems #graph-theory 

A graph G is [[k-connected|3-connected]] if and only if there exists a sequence of graphs $G_0,G_1,...,G_k$, such that 
- $G_0 = K_4$, 
- for each $i = 1,...,k$ the graph $G_i$ has two adjacent vertices $x^′,x^{′′}$ of [[degree]] ==at least 3==, so that $G_{i−1} = G_i ◦ x^′x^{′′}$, and $G_k = G$

ref: [[complete graph]], [[contract]]
